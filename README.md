# 半导体芯片缺陷检测系统

## 项目概述

本系统是一个用于半导体芯片AOI（自动光学检测）缺陷图像分类和检测的综合工具，提供了从图像预处理、数据标注到模型训练和预测的完整解决方案。系统采用PyQt5构建图形界面，使用PyTorch深度学习框架进行模型训练，支持多种预训练模型和数据增强方法。系统采用模块化设计，各组件通过Qt信号槽机制进行通信，具有良好的扩展性和维护性。

## 功能特点

### 1. 图像预处理
- 批量调整图像尺寸和格式
  - 支持自定义尺寸（如224x224、256x256等）
  - 支持多种输出格式（JPG、PNG、BMP等）
  - 使用LANCZOS算法进行高质量图像缩放
- 图像增强
  - 亮度调整（-50到50的调整范围）
  - 对比度调整（-50到50的调整范围）
  - 自动优化处理参数
- 数据集创建
  - 自动划分训练集和验证集（可配置比例）
  - 自动创建类别目录结构
  - 支持多级数据增强：
    - 基础增强：随机旋转90度、水平翻转、亮度对比度调整
    - 中等增强：基础增强 + 垂直翻转、高斯噪声、高斯模糊
    - 强化增强：中等增强 + 锐化、色调调整、弹性变换
  - 自动生成增强后的图像（每张原始图像生成3个增强版本）

### 2. 图像标注
- 支持多种标注工具
  - LabelImg：用于矩形框标注
    - 自动安装（如未安装）
    - 支持预定义类别文件生成
    - 自定义输出目录
    - XML格式标注文件
  - LabelMe：用于多边形标注
    - 自动安装（如未安装）
    - 自定义标注配置（颜色、形状等）
    - JSON格式标注文件
    - 支持多种标注形状（多边形、矩形、圆形、线段、点）
- 标注工具特性
  - 自动生成预定义类别文件
  - 自定义标注颜色和填充样式
  - 支持批量标注
  - 标注结果自动保存
- 标注格式转换（开发中）
  - 支持VOC格式
  - 支持COCO格式
  - 支持YOLO格式

### 3. 模型训练
- 支持多种深度学习任务
  - 图像分类
    - ResNet18/ResNet50
    - EfficientNet系列
    - 自定义网络结构
  - 目标检测（规划中）
    - YOLOv5
    - Faster R-CNN
    - SSD
- 训练功能
  - 支持GPU自动加速
  - 自动加载预训练模型
  - 实时训练监控
    - 损失值曲线
    - 准确率曲线
    - 学习率变化
    - 训练样本可视化
  - TensorBoard集成
    - 模型结构可视化
    - 训练指标记录
    - 样本图像展示
- 可配置参数
  - 基础参数
    - 批次大小（Batch Size）
    - 学习率（Learning Rate）
    - 训练轮数（Epochs）
  - 高级参数
    - 优化器选择（Adam/SGD/RMSprop）
    - 学习率调度器
    - 权重衰减
    - 是否使用预训练模型
- 评估指标
  - 分类任务
    - 准确率（Accuracy）
    - 精确率（Precision）
    - 召回率（Recall）
    - F1分数
  - 检测任务（规划中）
    - mAP
    - mAP_50
    - mAP_75
    - FPS

### 4. 缺陷预测
- 预测功能
  - 单张图像预测
  - 支持批量预测
  - 自动加载训练好的模型
  - 自动加载类别信息
- 预测结果
  - 显示前3个最可能的缺陷类别
  - 显示每个类别的置信度（百分比）
  - 可视化预测结果
  - 支持结果导出
- 预处理集成
  - 自动调整输入图像尺寸
  - 自动进行标准化处理
  - 与训练时相同的预处理流程

## 系统要求

- Python 3.7+
- PyTorch 1.7.0+
- PyQt5 5.15.0+
- Windows/Linux/MacOS
- CUDA支持（可选，用于GPU加速）

## 安装说明

1. 克隆或下载本项目

2. 安装依赖包
```bash
pip install -r requirements.txt
```

3. 运行程序
```bash
python src/main.py
```

## 使用指南

### 1. 数据处理

1. 选择包含原始图像的源文件夹
2. 选择输出根目录（系统将自动创建子目录）
3. 设置图像处理参数：
   - 目标尺寸（224x224、256x256等或自定义）
   - 目标格式（JPG/PNG/BMP）
   - 亮度和对比度调整
4. 设置数据集参数：
   - 训练集比例（默认0.8）
   - 数据增强级别（基础/中等/强化）
5. 点击"开始处理数据"
6. 处理完成后，点击"进入图片标注"

### 2. 图像标注

1. 选择标注工具（LabelImg或LabelMe）
2. 添加缺陷类别（系统预设了常见的缺陷类型：划痕、污点、缺失、变形、异物）
3. 点击"启动标注工具"
4. 在打开的标注工具中标注图像
5. 保存标注结果
6. 标注完成后，可直接进入模型训练阶段

### 3. 模型训练

1. 选择任务类型（分类或检测）
2. 选择标注数据文件夹
3. 选择模型类型
4. 设置训练参数：
   - 批次大小
   - 学习率
   - 训练轮数
   - 优化器
   - 学习率调度器
   - 权重衰减
   - 是否使用预训练模型
5. 选择评估指标
6. 点击"开始训练"
7. 训练过程中可实时查看进度、损失曲线和评估指标

### 4. 缺陷预测

1. 选择要预测的图像或文件夹
2. 选择使用的模型
3. 设置预测参数（如置信度阈值）
4. 点击"开始预测"
5. 查看预测结果
6. 可选择导出预测结果

## 配置文件说明

系统使用YAML格式的配置文件（config/config.yaml）存储默认设置，包括：

- 缺陷类别列表
- 默认路径设置
- 预处理参数
- 训练参数

用户可以通过界面修改这些设置，修改后的设置会自动保存。

## 注意事项

1. 首次使用标注工具时，系统会自动安装相应的工具，请确保网络连接正常
2. 对于大量图像，预处理和训练可能需要较长时间，请耐心等待
3. 训练模型时，如有GPU将自动使用，可大幅提升训练速度
4. 建议使用统一尺寸和格式的图像进行训练，以获得更好的效果
5. 系统会自动保存训练过程中的最佳模型，可在预测时选择使用
6. 对于检测任务，建议使用LabelImg进行矩形标注；对于需要精确轮廓的场景，建议使用LabelMe进行多边形标注
